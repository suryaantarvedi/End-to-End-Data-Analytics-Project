# 📊 End‑to‑End Data Analytics Project  

This repository contains a real‑world, company‑level data analytics project showcasing the complete workflow: Python for data cleaning, SQL for business analysis, and Power BI for interactive dashboard design.  

### 🚀 Retail Customer Behavior Analytics  

An end‑to‑end analytics project built on a retail shopping trends dataset to analyze customer demographics, purchasing behavior, and subscription patterns.  

It includes:  
- 🐍 **Python Data Preparation**: Data loading, cleaning, and exploratory analysis using pandas  
- 🗄️ **SQL Business Analysis**: Advanced queries for revenue by age group, gender segmentation, discount dependency, and customer loyalty insights  
- 📊 **Power BI Dashboard**: Interactive visuals summarizing KPIs, category performance, and customer trends  
- 🎨 **Design & Layout**: Recruiter‑ready formatting with slicers for dynamic filtering (subscription, gender, category, shipping type)  
- 🔍 **Visuals & Charts**:  
   - Pie chart for subscription status  
   - Bar charts for revenue and sales by category  
   - Horizontal bar charts for revenue and sales by age group  
   - KPI cards for customer count, average purchase, and review rating  
- 🎛️ **Interactivity**: Slicers to drill down into customer segments and shipping preferences  
- 📑 **Business Recommendations**: Actionable insights on subscriptions, loyalty programs, discount optimization, and targeted marketing  
- 🖼️ **Documentation**: Project report and acknowledgment of learning resources
